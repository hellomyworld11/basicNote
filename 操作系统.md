###　第２天

常用汇编指令 和常用寄存器

```
 MOV 赋值  MOV AX,0，相当于 AX=0;
 MOV BYTE [678],123   [] 表示内存地址
 ADD  加法 ADD SI,1  -> SI += 1
 JMP 跳转到指定地址
 
CMP AL, 0  比较 AL == 0
JE fin    条件跳转 上调指令满足条件再跳转

 INT 中断
 
 AX——accumulator，累加寄存器
 CX——counter，计数寄存器
 DX——data，数据寄存器
 BX——base，基址寄存器
 SP——stack pointer，栈指针寄存器
 BP——base pointer，基址指针寄存器
 SI——source index，源变址寄存器
 DI——destination index，目的变址寄存器
 
```





判断大小端

```
字节序模式 
小端： 低地址存储低字节
大端： 低地址存储高字节
网络字节序 是大端
#include <iostream>
using namespace std;
int main()
{
	int i = 0x01020304;
	if( *((char*)&i) == 0x01 )
	  cout << "big" << endl;
	else
	  cout << "small" << endl;
	return 0;
}
```
